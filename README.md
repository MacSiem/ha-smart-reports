# HA Smart Reports

![HA Smart Reports Screenshot](screenshot.png)

Generate comprehensive analytics reports for your smart home. Track energy usage, device activity, and automation performance.

## Installation

### HACS (recommended)

1. Open HACS in Home Assistant
2. Go to **Frontend** section
3. Click the three dots menu > **Custom repositories**
4. Add `https://github.com/MacSiem/ha-smart-reports` as **Dashboard**
5. Install **HA Smart Reports**
6. Restart Home Assistant

### Manual

Copy the contents to `/config/www/community/ha-smart-reports/`
